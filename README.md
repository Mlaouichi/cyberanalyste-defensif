# Cyberanalyste-Défensif

Boîte à outils gratuite et ouverte pour la cyberdéfense — scripts, bonnes pratiques et veille sécurité, pensés pour être accessibles à tous.

## Objectif

Ce projet vise à rendre la cybersécurité plus accessible : des outils simples, du code défensif (pas offensif), et une veille régulière sur l'actualité en cyberdéfense.

## Contenu

- **audit/** — Scripts d'audit (mots de passe faibles, ports ouverts, fichiers suspects)
- **bonnes-pratiques/** — Exemples de code sécurisé (hachage, validation, configuration)
- **pedagogique/** — Explications et démonstrations défensives (détection, prévention)
- **veille/** — Actualité et ressources en cyberdéfense, mises à jour régulièrement

## Pour qui

Formateurs, étudiants, organismes communautaires, développeurs — toute personne qui veut apprendre ou renforcer ses pratiques de sécurité.
